## การติดตั้ง G-Player Music Proxy Plugin

**ต้องมี backend proxy plugin สำหรับฟีเจอร์ค้นหาและเล่นเพลงออนไลน์**

---

### การติดตั้งแบบคลิกเดียว (แนะนำ)

> ⚠️ **ไปที่โฟลเดอร์หลักที่มีไดเรกทอรี SillyTavern ก่อนรันคำสั่งเหล่านี้**

#### 🪟 Windows (PowerShell)

1. เปิด PowerShell
2. ไปที่ตำแหน่ง SillyTavern เช่น:

   ```powershell
   cd D:\AI    # ถ้า SillyTavern อยู่ที่ D:\AI\SillyTavern
   ```

3. รันคำสั่งติดตั้ง:

   ```powershell
   cd SillyTavern && (Get-Content config.yaml) -replace 'enableServerPlugins: false', 'enableServerPlugins: true' | Set-Content config.yaml && cd plugins && git clone https://github.com/Sanjs333/g-player-proxy.git && cd g-player-proxy && npm install && cd ../.. && echo "Installation complete! Please restart SillyTavern"
   ```

#### 🐧 Linux / macOS

1. เปิด Terminal
2. ไปที่ตำแหน่ง SillyTavern:

   ```bash
   cd ~    # ถ้า SillyTavern อยู่ใน home directory
   ```

3. รันคำสั่งติดตั้ง:

   ```bash
   cd SillyTavern && sed -i 's/enableServerPlugins: false/enableServerPlugins: true/' config.yaml && cd plugins && git clone https://github.com/Sanjs333/g-player-proxy.git && cd g-player-proxy && npm install && echo "Installation complete! Please restart SillyTavern"
   ```

#### 📱 Termux (Android)

1. เปิด Termux
2. รันคำสั่งติดตั้ง:

   ```bash
   cd ~ && cd SillyTavern && sed -i 's/enableServerPlugins: false/enableServerPlugins: true/' config.yaml && cd plugins && git clone https://github.com/Sanjs333/g-player-proxy.git && cd g-player-proxy && npm install && echo "Installation complete! Please restart SillyTavern"
   ```

### การติดตั้งแบบ Manual

หากคำสั่งคลิกเดียวล้มเหลว ให้ทำตามขั้นตอนเหล่านี้:

**ขั้นตอนที่ 1: เปิดใช้ Server Plugins**

1. เปิด `config.yaml` ในไดเรกทอรีหลักของ SillyTavern
2. หา `enableServerPlugins: false`
3. เปลี่ยน `false` เป็น `true`
4. บันทึกไฟล์

**ขั้นตอนที่ 2: ค้นหาไดเรกทอรี Plugins**

หาโฟลเดอร์ `plugins` ในไดเรกทอรีหลักของ SillyTavern

**ขั้นตอนที่ 3: ดาวน์โหลด Plugin**

**ตัวเลือก A: ใช้ Git (แนะนำ)**

```bash
cd SillyTavern/plugins
git clone https://github.com/Sanjs333/g-player-proxy.git
cd g-player-proxy
npm install
```

**ตัวเลือก B: ดาวน์โหลด ZIP แบบ Manual**

1. ไปที่ [github.com/Sanjs333/g-player-proxy](https://github.com/Sanjs333/g-player-proxy)
2. คลิกปุ่มสีเขียว `Code` → `Download ZIP`
3. แตกไฟล์และเปลี่ยนชื่อโฟลเดอร์เป็น `g-player-proxy` วางใน `SillyTavern/plugins/`
4. เปิด terminal ในไดเรกทอรี plugin และรัน `npm install`

**ขั้นตอนที่ 4: รีสตาร์ท SillyTavern**

ปิดและรีสตาร์ท SillyTavern

## 🎨 ธีมตัวบ่งชี้

### ติดตามธีม UI หลักอัตโนมัติ

เมื่อเปิดใช้ "ซิงค์อัตโนมัติกับธีม UI หลัก" ธีมที่ตรงกับกฎการตั้งชื่อจะสลับโดยอัตโนมัติ

**กฎการตั้งชื่อ:** ชื่อธีม UI หลักต้องเป็น **คำนำหน้า** ของชื่อธีม/พรีเซ็ตตัวบ่งชี้

| ประเภท     | ตัวอย่างชื่อ                          |
| ---------- | --------------------------------- |
| ธีม UI      | PixelMint                         |
| พรีเซ็ตตัวบ่งชี้ | PixelMint                         |
| ธีมตัวบ่งชี้    | PixelMint-Style / PixelMint Theme |

### การจับคู่พรีเซ็ต-ธีมอย่างรวดเร็ว

เมื่อคุณเลือกพรีเซ็ตหรือธีม ปลั๊กอินจะแนะนำคู่ที่ตรงกันโดยอัตโนมัติ

**กฎการจับคู่:** ชื่อพรีเซ็ต + คำต่อท้าย = ชื่อธีม

คำต่อท้ายที่รองรับ: `-Style`, `Style`, `Theme`

| ชื่อพรีเซ็ต   | ชื่อธีมที่ตรงกัน                                            |
| --------- | ----------------------------------------------------- |
| PixelMint | PixelMint-Style, PixelMint Style, PixelMint Theme ฯลฯ |

## 🎵 เครื่องเล่น

> ⚠️ **ข้อจำกัดการค้นหาเพลง**
>
> การค้นหาเพลง**ไม่รองรับภาษาไทย**
>
> กรุณาใช้ชื่อเพลงในภาษาต้นฉบับ (จีน, อังกฤษ, ญี่ปุ่น, เกาหลี ฯลฯ) ในการค้นหา
>
> ตัวอย่าง: ใช้ "再见" แทน "ลาก่อน", ใช้ "Lemon" แทนชื่อที่แปลแล้ว

### ซ่อนอัตโนมัติ

เมื่อเปิด "ซ่อนอัตโนมัติ" เครื่องเล่นจะซ่อนตามค่าเริ่มต้น:

- **คลิกพื้นที่ว่างในแชท** → แสดง/ซ่อนเครื่องเล่น
- การคลิกปุ่ม ข้อความ หรือฟอง BGM จะไม่ทำให้เกิดการซ่อน
- เพลงยังเล่นต่อเนื่องเมื่อซ่อน โดยไม่สูญเสียความคืบหน้า
- เหมาะเมื่อไม่ต้องการให้เครื่องเล่นบังหน้าจอ

> อย่าเปิด "ซ่อนอัตโนมัติ" และ "โหมดพื้นหลัง" พร้อมกัน – ขัดแย้งกันทางตรรกะ

### โหลดเพลย์ลิสต์จาก World Book

เครื่องเล่นสามารถอ่านเพลย์ลิสต์ที่กำหนดใน **World Books ที่แนบกับตัวละคร (ไม่ใช่ Global World Books)** โดยอัตโนมัติ

**ขั้นตอนการตั้งค่า:**

1. สร้างรายการที่ **เปิดใช้งาน (ไฟเขียว)** ใน World Book
2. ตั้ง Key เป็น: `@BGM@`
3. กรอก Content ตามรูปแบบนี้:

```
title: All Alone With You
artist: EGOIST
audioUrl: https://…….mp3
coverUrl: https://…….gif
lyricsUrl: https://…….lrc
=====
title: Yasashii Suisei
artist: YOASOBI
audioUrl: https://…….mp3
=====
title: Close Your Eyes
artist: Honda Michiyo
audioUrl: https://…….mp3
coverUrl: https://…….jpg
lyricsUrl: https://…….lrc
```

**หมายเหตุรูปแบบ:**

- ใช้ **โคลอน `:`** ตามด้วยช่องว่าง
- แยกหลายเพลงด้วย `=====` (อย่างน้อย 3 เครื่องหมายเท่ากับ)
- URL ปกและ URL เนื้อเพลงเป็นตัวเลือกเสริม

### การเพิ่ม BGM อัตโนมัติ (เอาต์พุต AI)

หลังจากนำเข้า indicator World Book และเปิดใช้งานทั่วโลก แท็ก BGM ในการตอบกลับของ AI จะถูกเรนเดอร์เป็นฟองที่คลิกได้และเพิ่มลงในเพลย์ลิสต์โดยอัตโนมัติ

**รูปแบบเอาต์พุต AI:**

```
[bgm]ชื่อเพลง-ศิลปิน[/bgm]
```

> ⚠️ รูปแบบ BGM ต้องไม่ห่อด้วย \` หรือ \`\`\`

## ธีมไดนามิก

ธีมไดนามิกสามารถใช้สำหรับเอฟเฟกต์บรรยากาศ เมื่อ AI อธิบายสถานการณ์เฉพาะ การส่งออกแท็กทริกเกอร์จะสลับธีม

**รูปแบบทริกเกอร์:**

```
[Theme: ชื่อธีม]
```

**ตัวอย่าง:** เมื่อ AI อธิบายฉากหิมะตก การส่งออก `[Theme: 飘雪]` จะทริกเกอร์เอฟเฟกต์หิมะ
ระยะเวลาแสดงผลควบคุมโดยการตั้งค่า "ระยะเวลาธีม" (ตั้งเป็น 0 เพื่อซ่อนทันที)

**ธีมไดนามิกใช้ทรัพยากรและอาจทำให้แล็ก เปิดใช้ตามความสามารถของอุปกรณ์ของคุณ**

## ❓ คำถามที่พบบ่อย

### การติดตั้ง Plugin

**Q: คำสั่งคลิกเดียวล้มเหลว?**
A: ใช้การติดตั้งแบบ manual และทำตามขั้นตอนทีละขั้น

**Q: ไม่พบคำสั่ง `git` หรือ `npm`?**
A: ติดตั้ง [Git](https://git-scm.com/downloads) และ [Node.js](https://nodejs.org/) ก่อน

**Q: ไม่มีข้อความเริ่มต้น plugin หลังรีสตาร์ท?**
A: กรุณาตรวจสอบ:

1. `enableServerPlugins: true` ใน `config.yaml`
2. path plugin ถูกต้อง: `SillyTavern/plugins/g-player-proxy/index.js`
3. รัน `npm install` แล้วในไดเรกทอรี plugin

### ตัวบ่งชี้

**Q: ทำไมคลิกปุ่มภายในธีมไม่ได้?**
A: คุณต้องเช็ค "ล็อคตำแหน่ง" ในการตั้งค่าก่อน

**Q: สไตล์ตัวบ่งชี้เสีย?**
A: ตรวจสอบว่ามีธีมที่ตรงกันสำหรับพรีเซ็ตปัจจุบัน ลองใช้คู่ธีม-พรีเซ็ตที่ตรงกัน หรือใช้โหมดถาวรและรีเฟรชหน้า

**Q: ติดตามธีมอัตโนมัติไม่ทำงาน?**
A: ตรวจสอบว่าการตั้งชื่อถูกต้องตามกฎ—ชื่อธีม UI หลักต้องเป็นคำนำหน้าของชื่อธีมตัวบ่งชี้

**Q: รูปภาพไม่แสดงในตัวบ่งชี้?**
A: image hosting อาจถูกบล็อค ลองใช้ VPN และสลับโหนด (เช่น ญี่ปุ่น)

**Q: ตัวบ่งชี้เฉพาะตัวละครไม่ใช้งาน?**
A: หากคุณเลือกธีมจาก dropdown ทั่วไปด้วยตนเอง จะแทนที่การตั้งค่าเฉพาะตัวละครชั่วคราว

วิธีแก้ไข:

1. สลับไปแชทอื่นแล้วกลับมา (รีเซ็ตอัตโนมัติ)
2. เลือกธีมใหม่จาก dropdown

ลำดับความสำคัญ: การเลือกด้วยตนเอง > เฉพาะตัวละคร > ติดตาม UI > การตั้งค่าทั่วไป

### เครื่องเล่น

**Q: ฟอง BGM ไม่แสดง?**
A: ยืนยันว่าเอาต์พุต AI ใช้รูปแบบถูกต้อง `[bgm]เพลง-ศิลปิน[/bgm]` และไม่ห่อด้วย code blocks

**Q: ธีมเครื่องเล่นไม่แสดงในรายการ?**
A: ธีมเครื่องเล่นต้องมี `Player` เป็นคำนำหน้าชื่อเพื่อให้รู้จัก

**Q: AI ไม่ส่งออก BGM หรือธีมไดนามิก?**
A: ตรวจสอบว่า **indicator World Book เปิดใช้งานทั่วโลก** หรือ **ปรับตำแหน่งรายการ** รายการ BGM สามารถตั้งเป็น **Depth 0** ธีมไดนามิกเป็น **@⚙, Depth 0** หรือ **แก้ไขเนื้อหา prompt ของรายการ**

**Q: BGM ไม่เล่นในเครื่องเล่น?**
A: ลอง **รีสตาร์ทเครื่องเล่น** หรือ **รีเฟรชหน้า SillyTavern** ถ้ายังไม่เล่น เพลงอาจไม่พร้อมใช้งาน

**Q: ฟองเพลงควบคุมการเล่นเครื่องเล่นไม่ได้?**
A: ฟองใช้การจับคู่แบบคลุมเครือเพื่อระบุแทร็กปัจจุบัน หากควบคุมล้มเหลว ชื่อเพลง/ศิลปินในฟองอาจแตกต่างจากที่เครื่องเล่นแสดงมาก

**Q: ทำไมเครื่องเล่นไม่มีฟังก์ชันค้นหา?**
A: ใช้หน้าเครื่องมือเพื่อกู้คืนรายการในตัวเพื่ออัปเดตเครื่องเล่น หากคุณแก้ไขธีมในตัวและต้องการเก็บไว้ อย่าลืมส่งออกสำรองก่อน

**Q: ค้นหาเพลงไทยไม่เจอ?**
A: การค้นหาเพลงไม่รองรับภาษาไทย กรุณาใช้ชื่อเพลงในภาษาต้นฉบับ (จีน, อังกฤษ, ญี่ปุ่น, เกาหลี ฯลฯ)
